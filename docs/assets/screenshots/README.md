# Screenshots

This directory contains screenshot assets referenced by the documentation pages.

Screenshots are generated by the `recordDocsScreenshots` Gradle task using Roborazzi (preferred) or Paparazzi.

## Naming Convention

```
{page-id}_{description}.png
```

Examples:
- `onboarding_welcome.png`
- `connections_bluetooth_scan.png`
- `messages-and-channels_channel_list.png`
- `nodes_detail_metrics.png`

## Generation

```bash
./gradlew recordDocsScreenshots
```

## Guidelines

- PNG format, @2x resolution
- Max width: 1080px
- Prefer dark and light mode variants when relevant
- Name screenshots to match the page they appear in
