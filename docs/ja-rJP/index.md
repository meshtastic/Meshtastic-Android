# Meshtastic Android アプリのドキュメント

KMP（Kotlin Multiplatform）で構築された Meshtastic の Android、デスクトップ、iOS アプリ向けの、ユーザーおよび開発者向けドキュメントです。

サイドバーのナビゲーションから、アプリの機能については**ユーザーガイド**を、プロジェクトへの貢献については**開発者ガイド**をご覧ください。
