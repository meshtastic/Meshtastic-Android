# Skill: Meshtastic Design Standards

## Description
Brand identity, color palette, accessibility, and icon specifications for Meshtastic clients.

**Before making any UI changes**, fetch the authoritative, always-up-to-date design standards from the upstream repository:

- **Rendered (GitHub):** <https://github.com/meshtastic/design>
- **Raw markdown:** <https://raw.githubusercontent.com/meshtastic/design/refs/heads/master/standards/meshtastic_design_standards_latest.md>

The upstream document is the single source of truth and covers:
- Brand colors (primary `#2C2D3C`, accent `#67EA94`)
- Extended neutral and green scales with usage context
- Semantic colors (info, warning, error)
- WCAG AA accessibility guidance for light and dark modes
- Android launcher and action bar icon specifications
